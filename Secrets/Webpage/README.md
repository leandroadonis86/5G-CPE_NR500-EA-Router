# Webpage

In this router is possible to allow hidden debug menus to show up with an embedded account "root" added by manufactory and it will bring "Engineering Setting" menu. This account unfortually is available in all devices and the password can only be changed by modifying the jbc_settings.ini file with root access in shell.
